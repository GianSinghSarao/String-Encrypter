String Encrypter

A small single file program for encrypting strings of characters.

Initially, The result of following an ancient tutorial (Hooray for backwards compatibility!) about HTML Applications, this experiment has grown more than I thought possible.

Disclaimer:

String Encrypter, Despite its Name, Is Not Secure. It Is Not To Be Used In Any Place Where Any Form Of Sensitive Data Is Being Transferred Or Stored. It Is Easy To Break And Can Be Broken Quickly Using A Simple Brute-Force Attack.

Usage:

Download the latest version of the program from Here or use a heavily modified hosted version which works in most browsers Here

Open the file in the built-in program "mshta.exe". (Double clicking in Windows Explorer does this automatically)

Type a string into the main textarea and a password into the password field.

Click encrypt or decrypt.

By the way, the password used in the example images was 12345678. So you can type in the output of the encryption (D9L0S0S0V0P7S9V0Y0S0K0) and the password to decrypt it for yourself.

Disclaimer:

Because the String Encrypter is an HTA, which is obsolete and has been deprecated by newer techniques, it might not work on newer versions of windows. But more importantly, because it is propietary, it will not work at all on non-windows operating systems.
